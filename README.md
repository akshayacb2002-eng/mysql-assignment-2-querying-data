# MySQL Assignment 2 – Querying Data

## Overview
This project demonstrates SQL querying techniques using an Employee Management Database in MySQL. It covers data retrieval, filtering, sorting, grouping, aggregate functions, record updates, and table joins.

## Tools Used
- MySQL Workbench
- MySQL

## SQL Concepts Covered
- DISTINCT
- WHERE
- ORDER BY
- LIMIT
- Aggregate Functions (SUM, MIN, MAX, AVG)
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- UPDATE
